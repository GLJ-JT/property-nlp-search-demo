Launch sequence

Smarter than ever, without sounding technical.

This onboarding introduces the feature as a calmer, clearer search experience. It explains the value in user language, then shows the product behavior with motion and intent chips.

1. Smarter search Lead with the benefit: renters can type how they speak and see useful matches faster.
2. Editable intent Show the extracted filters so the experience feels transparent, not magical.
3. Explainable results Help the user understand why a home ranked where it did, so the flow feels trustworthy.
4. Actionable CTA Invite users into the live search rather than ending on a marketing screen.
